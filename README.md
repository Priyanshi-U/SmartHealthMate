# Smart HealthMate — AI Wellness Companion

## Overview
Smart HealthMate is an AI-based healthcare assistant designed for educational and research purposes. The project predicts possible diseases based on user symptoms using Machine Learning and provides basic health suggestions.

⚠ This project is currently under development. The repository represents only a glimpse of the complete Smart HealthMate system. I am continuously improving the model, features, and architecture.

## Features
- Voice-based symptom input using speech recognition  
- Disease prediction using Naive Bayes classification  
- Risk score gauge visualization  
- Basic medical suggestions and lifestyle tips  
- Conversation-style AI interaction  
- History tracking system  

## Technology Stack
- Python  
- Tkinter GUI  
- Natural Language Processing (NLTK)  
- Machine Learning (Scikit-learn)  
- Matplotlib Visualization  

## Dataset
Public healthcare symptom datasets are used for model training.
